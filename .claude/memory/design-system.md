Design wurde in REN-48 vom Nutzer abgenommen (2026-05-24). Vorher von Claude auto-generiert und nie genehmigt — jetzt explizit bestätigt.

**Typografie:** DM Sans, vollständig serifenlos. Kein zweiter Font.

**Farben:**
- Akzent: `#dc2626` (Rot)
- Dunkel: `#0f0f0f`
- Hell: `#f9fafb`
- Weiß: `#ffffff`
- Text gedämpft: `#6b7280`

**Layout Hero:** Split — Text/CTA links, dunkles App-Mockup rechts.

**Logo:** Rote Kreismarke mit Atemschutzmaske (`logo.png` im Repo-Root). Icon links neben Wortmarke, gap 0, Icon 64px, Schriftzug 1.375rem/800.

**Wortmarke:** `Atemschutz` (dunkel) + `buch` (#dc2626).

**Footer:** `Made with ❤️ in Lemgo` — kein weiterer Inhalt.

**Stimmung:** Modern + Zuverlässig. SaaS-artig, kein Klischee-Feuerwehr-Look.

**Why:** Der ursprüngliche Vorschlag (Fraunces + Source Serif 4, #CC2222) wurde nie vom Nutzer genehmigt. Kompletter Neustart in REN-48.

**How to apply:** Vor jeder Design- oder CSS-Entscheidung diese Tokens verwenden. Keine anderen Fonts oder Farben einführen ohne Rückfrage.
