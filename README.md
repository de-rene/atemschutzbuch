# Atemschutzbuch

Ein digitales Atemschutzbuch für kleine Freiwillige Feuerwehren.

## Worum geht's?

Der Atemschutznachweis – Einsätze, Übungen, Unterweisungen, Belastungsübungen und
Tauglichkeitsuntersuchungen jedes Atemschutzgeräteträgers (AGT) – wird in vielen
kleinen Wehren bis heute auf Papier oder in unübersichtlichen Tabellen geführt.
Das ist fehleranfällig, schwer auswertbar und macht es mühsam, Fristen im Blick zu
behalten.

Das **Atemschutzbuch** digitalisiert diesen Nachweis: übersichtlich, nachvollziehbar
und auf den Alltag kleiner Wehren zugeschnitten.

- **Zielgruppe:** kleine Freiwillige Feuerwehren.
- **Open Source & self-hosted:** der Code ist offen, die Daten bleiben bei der Wehr.
